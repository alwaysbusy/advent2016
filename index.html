Programming Advent Calendar 2016

More information

This advent calendar walks you through creating a festive e-card over 24 days. It will teach the basics of HTML, CSS, JavaScript and some popular tools used for front-end web development.

You can complete this using freely available tools:

  • A plain text editor - Atom or Visual Studio Code will work well
  • A web server - Python http.server is good for local, or GitHub Pages for sharing
  • Some photos - Your own or freely available online (the course covers this)

Each day links to code you can use to help you understand concepts further and develop your e-card. There are also links to websites which will help you learn more where appropriate.
